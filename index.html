Domínios WASH

O que avaliamos em cada área da unidade sanitária

O WASH FIT avalia 7 grandes áreas (domínios). O score de cada domínio mostra a percentagem de indicadores que cumprem os padrões mínimos. Abaixo de 67% é considerado nível limitado ou sem serviço pela OMS.

Indicadores

Estado actual de cada indicador por domínio WASH · escala 0–1–2

Cada indicador é avaliado em escala de 0 = Não cumpre (vermelho) · 1 = Cumpre parcialmente (amarelo) · 2 = Cumpre totalmente (verde). O semáforo grande mostra o nível geral do domínio com base nos filtros activos.

    Como ligar ao KoboToolbox

    1
    Aceda ao KoboToolbox → Account Settings → copie o API Token
    2
    Aceda ao formulário → Settings → copie o Asset UID do URL: kf.kobotoolbox.org/forms/ETwcIo5t...
    3
    Link de recolha do formulário: ee.kobotoolbox.org/x/ETwcIo5t — use para submeter dados de teste
    4
    Cole abaixo e clique Guardar e Testar. Os dados actualizam a cada 60 segundos
    !
    Nota CORS: Se aparecer "Failed to fetch", abra o dashboard num servidor HTTP local em vez de como ficheiro. Solução rápida:
    python -m http.server 8000 → abrir http://localhost:8000/washfitmoz.html
    5
    Campos mapeados do XLSForm: wscore sscore HCWMscore Hscore ECscore Escore Mscore finalscore → scores por domínio | provincia distrito unidade_sanitaria → filtros geográficos
